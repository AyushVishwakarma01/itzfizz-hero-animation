# Itzfizz Hero Animation

A scroll-based animated hero section built using HTML, CSS, and GSAP.

## Features
- Letter animation on load
- Scroll-triggered animation using GSAP
- Responsive layout
- Live deployed using GitHub Pages

https://ayushvishwakarma01.github.io/itzfizz-hero-animation/

## Tech Used
- HTML
- CSS
- JavaScript
- GSAP (ScrollTrigger)
